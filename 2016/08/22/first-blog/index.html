  1. 打开xf-adobecc2015.exe 注册机，生成序列号（请勿关闭注册机）

安装 ADOBE CC 软件：

  1. 点击 安装 我已经购买
  2. 点击 登录 ADOBE ID （确保已经断网）
  3. 选择 稍后连接
  4. 接受 许可协议，输入注册机 Gen.serial 生成的序列号，
  5. 点击 下一步，出现错误提示 “请连接到互联网，然后重试” ，点击 稍后连接。
  6. 进行安装
  7. 安装完成，点击 立即启动。（依旧断网状态）
  8. 点击 接到互联网时是否出现问题？ 链接
  9. 点击 脱机激活
  10. 点击 生成请求码
  11. 复制 请求代码 到注册机 Request code 选项中
  12. 复制注册机 Gen.activation 生成的新代码到 响应代码 中

  13. 激活成功，点击 启动一下 或 关闭

  14. 完成安装后，请双击 disable_activation.cmd （以管理员身份运行）。
